Installations:

1. For running programs in java, one has to install Java and an IDE for Java.
We have used Java version 1.8.0
For IDE, we have taken Eclipse

Following are the links for downloads respectively.

https://java.com/en/download/
http://www.eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/marsr

2. For runnning python scripts, download python and an IDE for python
We have used Python version 2.7.12
Anaconda is to be installed - version 4.2.0 - this is a package manager
IDE - Ipython notebook

Following are the links for downloads.

https://www.python.org/downloads/
https://ipython.org/ipython-doc/2/install/install.html

Once the installations are complete.
Run the 3 java programs - SimpleForecastingChartDemo.java // for forecasting
		        - ForecastingChartDemo.java // for normalizing in min-max and forecasting
		        - ZscoreForecastingChartDemo.java // for normalizing in z score and forecsting 

One can also directly import the executable jar file in eclipse.

Run python scripts - persistence.py // for forecasting using persistence model
		- ARIMA.py // for forecasting using ARIMA model
